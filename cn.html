Computer Networking Concepts

IP Address

Network Layer Unique Identifier

IP Address

A numerical label assigned to each device in a computer network that uses the Internet Protocol for communication.

  • Types: IPv4 (32-bit), IPv6 (128-bit)
  • Function: Identifies and locates devices on networks
  • Use case: Routing network traffic, device identification

Bit Stuffing

Data Link Layer Frame Synchronization

Bit Stuffing

A technique used in data transmission to prevent patterns in the transmitted data from being mistaken for control information.

  • Process: Insert extra bits into the data stream
  • Purpose: Ensure data transparency and maintain synchronization
  • Use case: HDLC, USB communication
